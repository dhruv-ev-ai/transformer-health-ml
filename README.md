## ⚡ AI Transformer Health Monitoring System

📊 Machine Learning project to predict transformer health condition using electrical parameters.

---

## 🚀 Project Overview

This project uses Machine Learning Classification to predict the condition of a transformer based on:

- 🌡 Temperature
- ⚡ Load Current
- 🔌 Voltage
- 🛢 Oil Level

The system classifies the transformer into:

- 🟢 Healthy
- 🟡 Warning
- 🔴 Faulty

An interactive interface is built using Gradio so users can enter values and instantly get predictions with visualization.
